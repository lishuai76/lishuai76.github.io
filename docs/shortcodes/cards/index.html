    Build-in figure shortcode #

    Hugo’s built-in figure shortcode is also styled as a card

    • {{< figure
  src="placeholder.svg"
  alt="A placeholder image"
  link="#"
  caption="A placeholder image caption"
  loading="lazy"
  target="_blank"
  title="Figure Title"
  caption="A caption for the figure"
  attr="An attribution"
  attrlink="#"
>}}
